# @nozomiishii/prettier-config

My personal [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
yarn add -D @nozomiishii/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@nozomiishii/prettier-config"
}
```
