## Install

```bash
pnpm add -D @nozomiishii/tsconfig
```

## Usage

`tsconfig.json`

base

```json
{
  "extends": "@nozomiishii/tsconfig"
}
```

nextjs

```json
{
  "extends": "@nozomiishii/tsconfig/tsconfig.nextjs.json"
}
```
